Object Explorer

Object Explorer is a module to check and update the current running live objects in the system.

Current version:

It works now with a shared folder.

1. Double click the ObjectExplorer.liveobject.

2. Drag and drop a shared folder with other liveobjects in it to the object explorer.

3. Click the tree node on the left tree view to expand the shared folder and explore the other liveobjects in it.
